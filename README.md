# Products Microservice

## Dev

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file in the root of the project based on the `.env.template` file
4. Execute prisma migrations with `npx prisma migrate dev`
5. Execute `npm run start:dev` to start the development server
